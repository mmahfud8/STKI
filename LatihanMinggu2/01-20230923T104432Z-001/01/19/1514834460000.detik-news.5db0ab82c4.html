Ini yang Diduga Jadi Penyebab Mobil Terbakar di Parkiran Merapi Park

Ristu Hanafi - detikNews
Share 0 Tweet Share 0 0 Komentar
Mobil terbakar di parkiran Merapi Park, Sleman. Foto: Ristu Hanafi/detikcom
0 0 0
Sleman - Sebuah mobil tiba-tiba terbakar di area parkir Merapi Park, Sleman siang tadi. Polisi menduga kebakaran akibat dari korsleting arus listrik.

"Api keluar dari bagian mesin depan ketika sopir menghidupkan mesin. Dugaan sementara akibat ada hubungan pendek arus listrik saat mesin di-starter," kata Kapolsek Pakem Kompol Haryanta, di lokasi kejadian kebakaran, Senin (1/1/2017).

Setelah api berhasil dipadamkan oleh petugas Pemadam Kebakaran Sleman, polisi langsung mengamankan bangkai mobil guna kepentingan olah TKP yang terletak di Desa Hargobinangun, Kecamatan Pakem, Sleman.

"Kita sudah mintai keterangan dari sopir, saksi, dan olah TKP. Dugaan sementara itu (korsleting listrik)," jelasnya.

Sore tadi, sebuah mobil sedan terbakar hebat di tempat parkir kendaraan Merapi Park. Tidak ada korban jiwa dalam peristiwa ini karena sopir dan 3 orang penumpang berhasil keluar dari mobil sebelum api membesar.

"Saat saya hendak menyalakan mesin, tiba-tiba ada suara letupan dari bagian mesin depan. Lalu keluar asap dan api," kata pemilik mobil, Farid warga Solo, di temui di lokasi kejadian.

Saat itu, dia bersama keluarganya hendak pulang setelah berwisata di Merapi Park. Di dalam mobi dia bersama tiga orang anggota keluarga.

"Setelah terlihat asap mengepul dan kobaran api, saya dan keluarga langsung keluar mobil," jelasnya.

Diakuinya, tidak ada keluarganya yang luka akibat peristiwa ini. Sesaat setelah menyelamatkan diri, kobaran api semakin membesar dan meludeskan seluruh badan mobil. Farid juga menduga pemicu api akibat korsleting listrik.
